# 校园书递

校园二手教材交易平台：Vue 3 + Flask + SQLite/MySQL。

## 启动

| 场景 | 方式 |
|------|------|
| 开发 | 双击 `run.bat` 或 `cmd /c run.bat` → http://localhost:5173 |
| 生产 | 双击 `本机部署.bat` → http://127.0.0.1:5000 |
| 测试 | `python -m pytest tests/ -q` |
