# 快速启动指南

## 🚀 一键启动（推荐）

### Windows 用户
直接双击 `setup_and_run.bat` 文件，或在命令行运行：
```bash
setup_and_run.bat
```

### Linux/Mac 用户
在终端运行：
```bash
chmod +x setup_and_run.sh
./setup_and_run.sh
```

### 跨平台（Python）
```bash
python setup_and_run.py
```

## 📋 脚本功能

运行脚本会自动完成以下操作：

1. ✅ **检查依赖** - 自动检测并安装所需依赖包
2. ✅ **初始化数据库** - 创建 SQLite 数据库和表结构
3. ✅ **迁移数据** - 检测并询问是否迁移现有 JSON 数据
4. ✅ **启动应用** - 自动启动 Flask 应用

## 🎯 默认账户

首次运行后，系统会自动创建以下账户：

**管理员账户：**
- 用户名：`admin`
- 密码：`admin123`

**测试账户：**
- 用户名：`student1`
- 密码：`student123`

## 🌐 访问应用

启动成功后，在浏览器访问：
- 主页：http://localhost:5000
- 登录页：http://localhost:5000/login
- 管理后台：http://localhost:5000/admin

## ⚠️ 注意事项

1. 首次运行会自动创建数据库文件 `campus_book_delivery.db`
2. 如果检测到 JSON 数据文件，脚本会询问是否迁移
3. 按 `Ctrl+C` 可以停止应用
4. 如果遇到问题，请查看 `SQLITE_MIGRATION_README.md` 获取详细说明

## 🔧 手动运行（可选）

如果一键脚本无法运行，可以手动执行以下步骤：

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 初始化数据库
python init_db.py

# 3. 迁移数据（可选）
python migrate_json_to_sqlite.py

# 4. 运行应用
python app.py
```
